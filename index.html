如何解决使用ThinkPHP框架开发网站不能正常显示验证码图片

Mac笔记本上来后出现验证码不能正常显示Fatal error: Call to undefined function Think\imagettftext()...

Download this project as a .zip file Download this project as a tar.gz file

初次使用ThinkPHP搭建而成的ThinkCMFX内容管理系统搭建网站时，创建完的项目后打开发现不能正常显示验证码图片。查看错误提示Fatal error: Call to undefined function Think\imagettftext()。后来发现原来是是因为MAC OSX 10.10 自带的PHP的GD库里缺少了PNG Support和FreeType Support，所以不能正常显示png图片。
解决办法如下:
在终端里输入以下代码
curl -s http://php-osx.liip.ch/install.sh | bash -s 5.6
等待完成后就能再查看phpinfo时就发现GD库已经成功安装所需要的组件了，再打开网站发现验证码就能正常显示了。
